This webpage uses HTML, CSS, and JavaScript. This website shows the deal of the day and has links for the rest of the week. The links use query strings to link that days coffee special. In the switch statement there is an object named coffee in each of the cases this is where the images, alt text, description, day, color, and name belong to. This is returned using a function and template literals on the webpage.
